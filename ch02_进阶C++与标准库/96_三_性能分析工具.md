## 三、性能分析工具

### 3.1 perf（Linux 性能分析器）

```bash
# CPU 性能分析（热点函数）
perf record -g ./myapp
perf report

# 缓存命中率
perf stat -e cache-references,cache-misses ./myapp

# 系统调用统计
perf stat -e syscalls:sys_enter_* ./myapp
```

### 3.2 Google Benchmark

```cpp
#include <benchmark/benchmark.h>

static void BM_VectorPushBack(benchmark::State& state) {
    for (auto _ : state) {
        std::vector<int> v;
        for (int i = 0; i < state.range(0); ++i) {
            v.push_back(i);
        }
        benchmark::DoNotOptimize(v);
    }
}
BENCHMARK(BM_VectorPushBack)->Arg(1000)->Arg(10000);
BENCHMARK_MAIN();
```

```bash
g++ -O3 -lbenchmark -lpthread bench.cpp -o bench
./bench
```

### 3.3 Valgrind

```bash
# 内存泄漏检测
valgrind --leak-check=full --show-leak-kinds=all ./myapp

# 性能分析（cachegrind）
valgrind --tool=cachegrind ./myapp
cg_annotate cachegrind.out.*
```
