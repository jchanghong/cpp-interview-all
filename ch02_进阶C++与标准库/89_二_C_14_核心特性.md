## 二、C++14 核心特性

### 2.1 泛型 Lambda

```cpp
auto add = [](auto a, auto b) { return a + b; };
add(1, 2);       // int
add(1.0, 2.0);   // double
```

### 2.2 函数返回类型推导

```cpp
auto add(int a, int b) { return a + b; }  // 推导为 int
```

### 2.3 `std::make_unique`

```cpp
auto p = std::make_unique<int>(42);  // C++14 引入
```

### 2.4 二进制字面量和数字分隔符

```cpp
int x = 0b1010;      // 二进制字面量
int y = 1'000'000;   // 数字分隔符（提高可读性）
```

### 2.5 变量模板

```cpp
template<typename T>
constexpr T pi = T(3.1415926535897932385);
```
