# cppinter — C++ 社招面试资料库（中国招聘市场）

## 仓库性质

C++ 社招面试准备资料库，**非代码仓库**。所有内容已合并为各分章节 Markdown 文件（代码见仓库根目录），另有合并版 `mini_all.md`（约 7,100 行），总计约 29,000 行

## 关键约束

1. **跨源合并** — 内容来自多个开源仓库合并（huihut/interview、青松与桑叶教程等），存在重复、分类不当等情况
2. **断裂链接** — URL 编码的相对链接（`problems__*.md`、`docs__*.md` 等）全部无效，修改时只写纯文本标题
3. **非 C++ 内容** — "面试鸭"React+Node.js 项目已移除；算法题解已清理 Python/Java 引用说明
4. **百度网盘链接** — 涉及版权风险，谨慎处理
5. **STL 章节** — 早期来自 cplusplus.com API 搬运，深层原理在后补章节中
6. **单例示例** — 懒汉式非线程安全，已加警告注释

## 修改规范

- 知识性修改须核实（优先 cppreference，非 CSDN）
- 新增内容用 `---` 分隔，保持 `#` / `###` 标题层级
- C++ 版本标注精确（如 `random_shuffle` C++14 废弃 C++17 移除）
- 保留旧文件名式标题（如 `# problems__xxx.md`）作为合并来源标识
- 对过时 API / 不安全代码补注释而非直接删除
