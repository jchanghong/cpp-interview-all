## 一、C++11 核心特性

### 1.1 自动类型推导（auto）

`auto` 让编译器在编译期根据初始化表达式推导变量类型。

```cpp
auto i = 42;           // int
auto d = 3.14;         // double
auto s = "hello";      // const char*
auto v = std::vector<int>{1, 2, 3};  // std::vector<int>

// 遍历容器时特别方便
for (auto& item : v) { std::cout << item << '\n'; }
```

**注意**：`auto` 推导时会忽略顶层 const 和引用，需要显式指定：
```cpp
const int ci = 10;
auto x = ci;        // x 是 int，const 被丢弃
const auto y = ci;  // y 是 const int
auto& z = ci;       // z 是 const int&
```

### 1.2 范围 for 循环（Range-based for）

```cpp
std::vector<int> v = {1, 2, 3, 4, 5};
for (const auto& x : v) { std::cout << x << '\n'; }
for (auto& x : v) { x *= 2; }  // 可修改
```

### 1.3 右值引用与移动语义

**左值 vs 右值**：
- 左值：有名称、可取地址的表达式（如变量 `x`）
- 右值：临时对象、字面量、不可取地址的表达式（如 `x + 1`、`std::move(x)`）

**移动构造函数**：窃取源对象资源而非拷贝
```cpp
class Buffer {
    char* data_;
    size_t size_;
public:
    // 移动构造函数
    Buffer(Buffer&& other) noexcept
        : data_(other.data_), size_(other.size_) {
        other.data_ = nullptr;
        other.size_ = 0;
    }
    // 移动赋值运算符
    Buffer& operator=(Buffer&& other) noexcept {
        if (this != &other) {
            delete[] data_;
            data_ = other.data_;
            size_ = other.size_;
            other.data_ = nullptr;
            other.size_ = 0;
        }
        return *this;
    }
};
```

**`std::move`**：将左值强制转换为右值引用，触发移动语义
```cpp
std::string s1 = "hello";
std::string s2 = std::move(s1);  // s1 被移走，处于有效但未指定状态
```

### 1.4 完美转发（Perfect Forwarding）

万能引用（`T&&`）+ `std::forward` 实现参数类型和值类别的无损传递：

```cpp
template<typename T, typename... Args>
std::unique_ptr<T> make_unique(Args&&... args) {
    return std::unique_ptr<T>(new T(std::forward<Args>(args)...));
}
```

**引用折叠规则**：
| 模板参数 | 实参类型 | 推导结果 |
|---------|---------|---------|
| `T&&` | `int&` | `int&` |
| `T&&` | `int&&` | `int&&` |
| `T&&` | `const int&` | `const int&` |

### 1.5 Lambda 表达式

```cpp
// 基本语法：[捕获列表](参数列表) mutable -> 返回类型 { 函数体 }
auto add = [](int a, int b) { return a + b; };

// 捕获方式
int x = 10;
auto l1 = [x]() { return x; };       // 值捕获
auto l2 = [&x]() { return ++x; };    // 引用捕获
auto l3 = [=]() { return x + 1; };   // 按值捕获所有可见变量
auto l4 = [&]() { ++x; };            // 按引用捕获所有可见变量
auto l5 = [x, &y]() { return x + y; }; // 混合捕获

// mutable：允许修改值捕获的变量
auto l6 = [x]() mutable { return ++x; };
```

### 1.6 智能指针（C++11）

- `std::unique_ptr`：独占所有权，不可拷贝，可移动
- `std::shared_ptr`：共享所有权，引用计数线程安全
- `std::weak_ptr`：弱引用，不增加引用计数，解决循环引用

> 详见上文「2.4 智能指针」章节。

### 1.7 并发编程基础（`<thread>`, `<mutex>`, `<condition_variable>`, `<future>`）

```cpp
// 线程创建
std::thread t([]{ std::cout << "hello from thread\n"; });
t.join();  // 等待线程结束

// 互斥锁
std::mutex mtx;
std::lock_guard<std::mutex> lock(mtx);  // RAII 加锁/解锁

// 条件变量
std::condition_variable cv;
std::unique_lock<std::mutex> lk(mtx);
cv.wait(lk, []{ return ready; });  // 等待条件满足
cv.notify_one();  // 唤醒一个等待线程

// future/promise
std::promise<int> p;
std::future<int> f = p.get_future();
std::thread t([&p]{ p.set_value(42); });
int result = f.get();  // 阻塞直到值就绪
t.join();
```

### 1.8 其他 C++11 重要特性

- **`nullptr`**：类型安全的空指针，替代 `NULL`/`0`
- **`enum class`**：强类型枚举，不隐式转换为 int
- **`static_assert`**：编译期断言
- **`override` / `final`**：虚函数重写标记和禁止重写/继承
- **`noexcept`**：异常规范，帮助编译器优化
- **`constexpr`**：编译期常量/函数
- **`std::array` / `std::tuple` / `std::unordered_map` / `std::unordered_set`**：新容器
- **`std::function` / `std::bind`**：通用函数包装器
- **`std::chrono`**：时间库
- **`std::regex`**：正则表达式
- **`std::initializer_list`**：初始化列表支持
