## 三、C++17 核心特性

### 3.1 结构化绑定（Structured Binding）

```cpp
std::map<std::string, int> m = {{"a", 1}, {"b", 2}};
for (const auto& [key, value] : m) {
    std::cout << key << ": " << value << '\n';
}

// 解构 tuple/pair/struct
auto [x, y, z] = std::make_tuple(1, 2.0, "hello");
```

### 3.2 `std::optional`

```cpp
std::optional<int> find(int key) {
    if (key > 0) return key * 2;
    return std::nullopt;
}

if (auto result = find(5)) {
    std::cout << *result << '\n';  // 10
}
```

### 3.3 `std::variant`（类型安全的 union）

```cpp
std::variant<int, double, std::string> v;
v = 42;
v = "hello";
std::cout << std::get<std::string>(v) << '\n';

// 访问
std::visit([](auto&& arg) { std::cout << arg << '\n'; }, v);
```

### 3.4 `std::any`

```cpp
std::any a = 42;
a = std::string("hello");
std::cout << std::any_cast<std::string>(a) << '\n';
```

### 3.5 `std::string_view`

零拷贝字符串视图，避免不必要的字符串拷贝：

```cpp
void print(std::string_view sv) {
    std::cout << sv << '\n';  // 不拷贝字符串数据
}
print("hello");           // const char* → string_view
std::string s = "world";
print(s);                 // std::string → string_view
print(s.substr(0, 3));    // 子串 → string_view（零拷贝）
```

### 3.6 `if` / `switch` 带初始化器

```cpp
if (auto it = m.find(key); it != m.end()) {
    std::cout << it->second << '\n';
}

switch (auto status = getStatus(); status) {
    case Status::OK: break;
    case Status::ERROR: break;
}
```

### 3.7 内联变量（Inline Variables）

```cpp
// 头文件中定义，无需担心 ODR 违规
inline constexpr double pi = 3.1415926535897932385;
```

### 3.8 文件系统库（`<filesystem>`）

```cpp
namespace fs = std::filesystem;
fs::path p = "/tmp/test.txt";
if (fs::exists(p)) { std::cout << "exists\n"; }
for (const auto& entry : fs::directory_iterator("/tmp")) {
    std::cout << entry.path() << '\n';
}
```

### 3.9 并行 STL 算法

```cpp
#include <execution>
std::vector<int> v(1000000);
std::sort(std::execution::par, v.begin(), v.end());  // 并行排序
```

### 3.10 折叠表达式（Fold Expressions）

```cpp
template<typename... Args>
auto sum(Args... args) {
    return (args + ...);  // 右折叠
}
```

### 3.11 `std::byte`

```cpp
std::byte b{0x0F};  // 类型安全的字节类型
```
