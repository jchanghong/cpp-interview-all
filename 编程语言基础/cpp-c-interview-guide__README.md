## 写在前面

大家好，我是青松与桑叶。写这个项目的初衷很简单——当年我自己准备 C/C++ 面试的时候，翻遍了网上的各种资料，发现一个很头疼的问题：市面上的八股文要么只有题目没有讲解，看完之后还是云里雾里；要么只有理论没有代码，看完之后根本不知道怎么用。更别提那些直接甩一堆标准答案让你背的，背完转头就忘。

所以我想做一个不一样的教程。我的教学理念很简单：**每个知识点 = 通俗讲解 + 示例代码 + 面试真题 + 踩坑提醒**。我不会上来就甩一堆专业术语吓你，而是先用大白话把概念讲清楚，再给你一段可以编译运行的代码，让你自己动手敲一遍。最后再告诉你面试官一般怎么考这个知识点，怎么回答才能拿高分。这样学下来，你不仅记住了，而且真正理解了。

和其他项目相比，本教程最大的特点就是"保姆级"——不会跳过任何中间步骤，不会假设你已经懂了什么前置知识。每一个概念都是从零开始讲起，代码里的每一行都有中文注释。如果你是零基础或者基础薄弱，跟着这个教程一步步学下来，面试的时候一定能自信地回答出来。

## 快速开始

每个知识点的代码都可以直接编译运行：

**C语言代码：**

```bash
gcc -Wall -o demo demo.c && ./demo
```

**C++代码：**

```bash
g++ -std=c++17 -Wall -o demo demo.cpp && ./demo
```

## 推荐学习资源

1. **《C Primer Plus》** — C语言入门经典，讲解非常详细，适合零基础
2. **《C++ Primer》** — C++入门圣经，内容全面，但比较厚，建议当工具书翻阅
3. **《Effective C++》** — C++进阶必读，55条具体建议，面试高频考点来源
4. **《STL源码剖析》** — 侯捷老师经典之作，深入理解STL底层实现
5. **《深入理解计算机系统》（CSAPP）** — 理解计算机底层原理，面试操作系统类问题的必备
6. **《Unix环境高级编程》（APUE）** — Linux/Unix编程圣经，网络编程和系统编程必读
7. **侯捷老师的C++视频课程** — B站免费资源，讲解生动，强烈推荐
8. **cppreference.com** — C/C++最权威的在线参考文档，遇到不确定的API随时查阅
9. **GDB调试技巧** — 学会用GDB调试程序，面试手撕代码时能快速定位问题
10. **LeetCode** — 算法题刷题平台，C/C++岗位面试必刷

## 常见问题

**Q：这个教程适合零基础吗？**

A：适合。本教程从最基础的概念讲起，不会跳过任何中间步骤。但如果你完全没有编程经验，建议先学一门简单的语言（比如Python）入门，再来学C/C++会轻松很多。

**Q：代码用什么编译器？**

A：教程中的代码使用 GCC/G++ 编译，C语言代码用 `gcc` 编译，C++代码用 `g++ -std=c++17` 编译。如果你用 Clang 或者 MSVC，大部分代码也能正常编译，个别地方可能需要微调。

**Q：需要按顺序学习吗？**

A：建议按顺序学习。第一部分是C语言基础，是后面所有内容的地基。如果你已经有一定基础，可以跳过熟悉的章节，直接看需要复习的部分。

**Q：面试只背这些够吗？**

A：八股文只是面试的一部分，除了基础知识，你还需要刷算法题（LeetCode）、准备项目经验介绍、了解目标公司的技术栈。本教程帮你打牢基础，让你在面试中不会因为基础问题丢分。

**Q：发现错误怎么办？**

A：欢迎提 Issue 或者 Pull Request，我会尽快修正。大家一起完善这个教程，帮助更多人。
