## 六、高性能网络编程要点

### 6.1 CPU Cache 友好性

- **避免 false sharing**：不同线程访问的变量放在不同 cache line（64 字节对齐）
- **数据局部性**：连续内存布局（如 `vector`）优于分散布局（如 `list`）
- **预取**：`__builtin_prefetch()` 提示 CPU 预取数据

### 6.2 连接管理

- **TIME_WAIT 处理**：使用 `SO_REUSEADDR` / `SO_REUSEPORT`
- **连接池**：复用 TCP 连接，减少握手开销
- **优雅关闭**：`shutdown(fd, SHUT_WR)` 通知对端，等待数据发送完毕再 `close()`

### 6.3 多线程模型

- **One Loop Per Thread**：每个线程一个 event loop（muduo 模型）
- **线程池 + 任务队列**：I/O 线程负责网络，Worker 线程负责计算
- **无锁队列**：线程间通信使用 lock-free queue 减少锁竞争
